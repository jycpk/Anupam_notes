# Computer Notes
### Useful Commands

- `cd` -> to change directory
- `mkdir` -> to make / create directory (folder)
- `ls` -> ls command shows list of files or directory
- `ls -l` it shows the long(detailed list)
- `pwd` it shows the location of present working directory
